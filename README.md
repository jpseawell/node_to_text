# Node to Text

## How to install in Blender

1. Open this GitHub page in your browser.
2. Click the [node_to_text.zip](./node_to_text.zip) file in the file list.
3. Click **Download raw file** to save it to your computer.
4. Open **Blender**.
5. Go to **Blender > Preferences > Add-ons**.
6. Click **Install...**
7. Select the downloaded [node_to_text.zip](./node_to_text.zip) file.
8. Turn on **Node to Text** in the add-ons list.

## How to use it

1. Open a node editor in Blender.
2. Press **N** to open the side panel if needed.
3. Click the **Node to Text** tab.
4. Use:
   - **Export Current Graph**
   - **Import From Clipboard**
